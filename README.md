# 📚 Student Study Planner Agent

An AI-powered study assistant built using Streamlit and Google Gemini API.  
It helps students generate study materials, create schedules, summarize notes, and track learning progress.

---

## 🚀 Features

### 📖 Fetch Study Materials
- Get AI-generated resources for any subject and topic
- Includes explanations and structured learning flow

### ⏰ Schedule Tasks
- Create personalized study schedules
- Based on hours, deadline, and priority

### 📝 Summarize Notes
- Paste long notes and get concise summaries
- Generates flashcards for quick revision

### 📊 Track Progress
- Analyze study progress
- Provides AI-based feedback and motivation

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Generative AI (Gemini)
- dotenv
