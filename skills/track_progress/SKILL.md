\# Track Study Progress Skill



\## Description

This skill monitors your study progress in real-time. It tracks completion percentage, achievements, compares performance against goals, and provides motivational feedback to keep you on track.



\## Input Required

\- \*\*Subject\*\*: The subject you're studying

\- \*\*Topic\*\*: Specific topic being tracked

\- \*\*Goals\*\*: Target completion percentage or time

\- \*\*Time Spent\*\*: Hours spent studying

\- \*\*Tasks Completed\*\*: Number of tasks/problems solved



\## Output Provided

\- Progress percentage (0-100%)

\- Completion timeline

\- Achievements unlocked

\- Performance insights

\- Motivational feedback

\- Comparison with goals

\- Saved to data/progress.json



\## How It Works

1\. User logs study session details

2\. Skill calculates progress percentage

3\. Compares against set goals

4\. Identifies milestones achieved

5\. Generates motivational insights

6\. Shows time remaining to goal

7\. Stores all data in JSON file



\## Example Usage



\*\*Example 1\*\*: Subject: Data Structures \& Algorithms

\- Topic: Sorting Algorithms

\- Goal: 100% completion in 2 weeks

\- Time Spent: 5 hours

\- Tasks Completed: 15/30 problems

\- Result: 50% progress, 7 days remaining



\*\*Example 2\*\*: Subject: Operating Systems

\- Topic: CPU Scheduling

\- Goal: 80% mastery in 3 weeks

\- Time Spent: 8 hours

\- Tasks Completed: 4/5 modules

\- Result: 80% progress, milestone achieved!



\*\*Example 3\*\*: Subject: Python Programming

\- Topic: Control Flow \& Loops

\- Goal: Complete in 1 week

\- Time Spent: 3 hours

\- Tasks Completed: 20/20 exercises

\- Result: 100% completed, 4 days early!



\*\*Example 4\*\*: Subject: Computer Networks

\- Topic: Network Layer

\- Goal: 70% understanding in 4 weeks
